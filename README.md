# SymmetryQNN
This is the register for the numerical simulation of Werner state example and 2D classification.

## Dependencies
Please download qiskit and pennylane for running codes. 

If you want to run the code of the second numerical results, say the $2\times 8$ qubits experiments, please switch to feature branch and run the main.jl in JuliaSupport folder. 

## how to run the code
The main entry of results are \*train.py in corresponding folder.
